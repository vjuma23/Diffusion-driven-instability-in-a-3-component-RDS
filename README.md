# Diffusion-Driven-Instability-in-a-3-component-RDS

This repository contains Python codes for analyzing and visualizing the parameter spaces of **three-component reaction-diffusion systems**. It specifically identifies regions where a steady state is stable without diffusion but becomes unstable when diffusion is introduced (DDI), leading to pattern formation.

## ðŸ”¬ Project Overview

The included scripts calculate stability conditions for reaction-diffusion models using various experimental and theoretical diffusion matrices we call polymer models.

### Key Features

- **Condition Mapping**: Evaluates traces, determinants, and discriminants of the system's Jacobian and diffusion matrices.
- **17-Panel figure**: Maps 17 distinct instability spaces (Space 1 through Space 99) representing various mathematical combinations of instability criteria.
- **High-Resolution Visualization**: Generates 3D scatter plots and coordinate data for further analysis.
- **Mode Isolation Analysis**: Computes characteristic polynomials and eigenvalue across modes / wavenumbers to isolate which spatial modes drive instability.

### `param_space_3_cmt.py`

The primary data generator:

- Iterates through a 3D meshgrid of parameters `(a, b, c)`.
- Applies stability logic for 11 different polymer diffusion models (one at a time).
- **Outputs**: Generates CSV coordinate files and individual 3D scatter plots for all spaces, including the "without diffusion" base stability region.

## ðŸš€ Getting Started

### Prerequisites

Ensure you have Python installed along with the required libraries:

```bash
pip install numpy matplotlib pandas
```

### Execution Instructions

#### 1) Generate the Data (DDI parameter spaces)

Run the generator script and specify the polymer model using the `--polymer` flag:

```bash
python param_space_3_cmt.py --polymer 4
```

Optional: Use `--res` to adjust grid resolution (e.g., `--res 100` for testing).

#### 2) Generate Visualizations (17-panel figure)

Create the multipanel comparison figure for the same polymer model:

```bash
python plot_from_csv_mult.py --polymer 4
```

#### 3) Run Mode Isolation (Q-polynomials + eigenvalue growth)

Run the mode isolation workflow by choosing a polymer model and a scale factor `gamma`:

```bash
python mode_isolate.py --polymer 6 --gamma 250
```

Optional parameters:

- `--a`, `--b`, `--c`: set the reaction parameters (defaults are built into the script for polymer 6)
